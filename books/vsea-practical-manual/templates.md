# テンプレート集

## テンプレート1: プロジェクト全体構成

```markdown
# [プロジェクト名] - VSEA開発計画

## プロジェクト概要
- 目的: [プロジェクトの目的]
- 期間: [開始日] - [終了予定日]

## フェーズ①: UX整合設計
- 期間: [日数]
- 成果物:
  - [ ] ユーザーストーリー
  - [ ] 画面遷移図

## フェーズ②: 共通スケルトン作成
- 期間: [日数]
- 成果物:
  - [ ] インターフェース定義

## フェーズ③: 縦スライス実装
### Slice 1: [機能名]
- 優先度: 高/中/低
- 期間: [時間]
```

## テンプレート2: スライス実装計画

```markdown
# Slice [番号]: [機能名]

## 概要
[目的と実現内容]

## 実装範囲
### UI層
- [ ] [画面1]

### ビジネス層
- [ ] [クラス].[メソッド]

## 完了判定
- [ ] 動作確認
- [ ] テスト実施
- [ ] デモ成功
```

## テンプレート3: デイリーログ

```markdown
# [日付] 開発ログ

## 今日の目標

## 実施内容
- [実装した内容]

## 問題と解決策
- 症状: [具体的な症状]
- 解決策: [採用した解決策]

## 明日の予定
- [ ] [タスク1]
```
